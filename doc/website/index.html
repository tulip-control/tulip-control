The Temporal Logic Planning (TuLiP) toolbox

summary

The Temporal Logic Planning (TuLiP) toolbox is a Python package for automatic synthesis of correct-by-construction embedded control software.

first steps

  1. Download the current release.
  2. To install it, try
    $ python setup.py install
    Detailed instructions are available.
  3. Explore the demonstrations under examples/, which are distributed with the release.
  4. Consult the User's Guide and API documentation.

going further

source repository

The source repository is available via Git:

$ git clone https://github.com/tulip-control/tulip-control.git
Beware that TuLiP is under active development, and the code in that repository may break without warning. Most users should try a release first.

bug reports

Bug reports and feature requests should be made by opening an issue on our project site at GitHub. The authors may also be contacted directly.

contributing

Our primary development site is hosted on GitHub. External contributions are welcome by:

  • Sending a patch by email.
  • Forking and submitting a pull request.
A general entry point for project planning is the developers' wiki.

contact and support

  • Join the tulip-control-announce mailing list.
  • Visit the #tulip-control channel of the freenode IRC network.
  • Contact project members at tulip@tulip-control.org
